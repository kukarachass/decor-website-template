# Žoria Decor — макет інтернет-магазину декору

Демо-макет магазину декору для дому в ніжній мінімалістичній естетиці.
Усі дані (товари, замовлення, користувачі, оплата) — **фейкові**, зберігаються
в `localStorage`. Бекенду немає.

## Стек

- **Next.js 15** (App Router, TypeScript)
- **Tailwind CSS v4** (дизайн-токени в `app/globals.css`)
- **Zustand** + `persist` — кошик, обране, авторизація, каталог і замовлення
- **lucide-react** — іконки
- Шрифт: **Inter** (400–800, кирилиця). `.font-display` — 800 для великих
  заголовків, `.font-title` — 600 для підзаголовків і карток, текст — 450
- **Двомовність UA / EN** — власний контекст `lib/i18n` + перемикач у шапці,
  футері та мобільному меню; вибір мови зберігається в `localStorage`

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # продакшн-збірка
```

## Мова

Словники — `lib/i18n/uk.ts` (джерело істини для типів) та `lib/i18n/en.ts`.
Переклади карток товарів — `lib/i18n/products.en.ts` (характеристики
перекладаються словником термінів). Компоненти беруть тексти через `useT()`,
товари — через `useLocalizedProduct()` / `useLocalizeProducts()`.

## Про зображення

Замість фото використовується компонент `components/ui/ProductArt.tsx`: він
генерує детерміновану ілюстрацію — мʼякий градієнт, підкладка-арка та лінійний
малюнок категорії (по 2–4 варіанти), тонований акцентним кольором категорії. Досить замінити цей компонент
на `next/image`, щоб підставити справжні знімки.

## Структура

```
app/
  (site)/          публічна частина (спільний хедер, футер, кошик-дровер, пошук)
  admin/           адмін-панель (власний лейаут і гард за роллю)
components/
  home/            секції головної
  catalog/         каталог і фільтри
  product/         картка товару, рейл, сторінка товару
  layout/          хедер, футер, дровери, форми
  ui/              кнопки, поля, бейджі, логотип, ProductArt
lib/
  data/            товари, категорії, доставка, мок-замовлення
  store/           zustand-стори
```

## Що варто памʼятати

- Дані персистяться в `localStorage` (`zoria-cart`, `zoria-auth`, `zoria-shop`,
  `zoria-wishlist`). Кнопка «Скинути демо» в адмінці повертає початковий каталог
  і замовлення.
- Оплата, SMS, накладні — імітація. Форма картки задизейблена навмисне.
